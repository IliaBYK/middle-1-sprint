Проект "Мессенджер" в рамках обучения в Я.Практикуме.
Стадия: **разработка**.

- `npm i` - устанавливает зависимости
- `npm run dev` - собирает проект в dev-режиме из src в dist
- `npm run start` - поднимает сервер на 3000 порту
